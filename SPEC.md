## 1. 서비스 개요

| 항목 | 내용 |
|---|---|
| 서비스명 | 가온이랑 시온이랑 놀자 |
| 대상 | 만 5~7세 유아 (주 사용자: 가온이·시온이) |

## 2. 핵심 원칙 (정책)

1. **유아 친화 톤** — 혼내지 않음. 오답엔 "다시 해볼까?" 같은 부드러운 격려 + 가벼운 효과음. 실패/게임오버도 위협적이지 않게.
2. **오프라인·무에셋 우선** — 콘텐츠는 가능한 한 이모지·CSS·합성음·직접 그린 SVG로 구성해 네트워크 없이도 동작하고 가볍게 유지. (예외: 국기 게임의 국기 이미지, 일부 카드 썸네일)
3. **서버리스·무수집** — 백엔드 없음. 개인정보를 서버로 전송하지 않음. 진행도는 **기기 로컬(localStorage)**에만 저장.
4. **성장·수집 동기** — 모든 게임이 동일한 틀(레벨 진행 → 별 1~3개 평가 → 저장 → 다음 단계) 위에서 돌고, 클리어 시 **스티커**를 모은다.
5. **개인화** — 플레이어(가온이/시온이)를 선택하면 이름·사진이 앱 전체에 반영되고, 스티커 도감은 플레이어별로 분리.
6. **접근성·단순함** — 큰 버튼/글씨, 손글씨풍 폰트, 명확한 색 대비. 한 화면에 한 가지 과제.

## 3. 기술 스택 / 아키텍처

- **프레임워크**: React 18 + TypeScript (strict)
- **빌드/번들**: Vite 6
- **라우팅**: React Router 7 (`createBrowserRouter`, `basename`으로 GitHub Pages 서브경로 대응)
- **스타일**: Tailwind CSS v4 (`@theme`로 커스텀 폰트 등록)
- **애니메이션**: Motion (Framer Motion)
- **효과음**: Web Audio API 합성 (외부 음원 파일 없음) + Web Speech(동물 소리/한글)
- **드래그**: react-dnd + TouchBackend (마우스+터치 통합)
- **연출**: canvas-confetti
- **배포**: GitHub Pages + GitHub Actions (`main` 푸시 시 자동 빌드·배포)

### 폴더 구조 (요약)
```
src/app/
├─ pages/        게임 12종 + Home + StickerBook
├─ components/   GameShell · ResultModal · StarRating · GameCard · Layout
└─ lib/          storage · useGameProgress · scoring · sound · celebrate ·
                 shuffle · useWrongFeedback · player · stickers · hangul · hiddenScenes
```

### 공통 컴포넌트/유틸 (재사용)
- `GameShell` — 모든 게임의 일관 프레임(배경 블롭·상단바·뒤로가기·제목·레벨 점·상태·다시하기)
- `ResultModal` — 클리어 공통 모달(별·플레이어 이름·**스티커 자동 지급**·다음단계/다시/홈)
- `useGameProgress(gameId)` — 레벨 이어하기 + 결과 저장
- `scoring` — `starsForLower`(실수·시간·걸음 적을수록↑), `starsForHigher`(점수·거리 클수록↑)
- `useWrongFeedback` — 공통 오답 피드백(효과음 + 격려 메시지)
- `sound` — 합성 효과음(정답/오답/팝/팡파레/레벨업) + `speak`(TTS)

## 4. 공통 시스템 정책

### 4.1 레벨 진행
- 게임마다 `LEVELS` 배열로 난이도 3단계(일부는 장면 6단계). 클리어 시 ResultModal "다음 단계"로 점점 어려워짐.
- 마지막 단계는 반복하며 별/스티커를 더 모을 수 있음. 도달 레벨은 저장되어 다음 방문 시 이어서.

### 4.2 별점 (1~3개)
- 게임마다 정해진 지표로 평가(아래 게임표 참고). 클리어는 **최소 1별** 보장.
- 게임별 **최고 별점**만 저장(낮아져도 기록 유지).

### 4.3 스티커 (수집 보상)
- **이모지 스티커, 7개 앨범 총 42종**: 🐾동물친구들 · 🍓맛있는간식 · 🌊바다친구들 · 🚀우주탐험 · 🦕공룡나라 · 🌈자연과날씨 · ✨반짝이(특별)
- **지급 규칙**: 게임 클리어 시 **별 개수만큼**(1~3장) 지급, **아직 안 모은 새 스티커 우선**. **3별이면 반짝이 스티커 1장 추가**.
- **플레이어별 분리** 도감. 모은 건 컬러, 안 모은 건 흐릿한 "?", 중복은 갯수 표시.
- 상단 스티커북 버튼(진행도 뱃지) → `/stickers` 도감 페이지.

### 4.4 플레이어
- **가온이 / 시온이** 중 선택(홈 상단). 선택 이름은 헤더·인사말·설명·성공화면에 반영("○○야, 어떤 놀이를 시작해볼까?", "○○랑 재미있게 놀아보아요!").
- 선택 버튼엔 실제 아이 사진(원형). 두 이름 모두 "이"로 끝나 한국어 조사가 일관(○○가/○○랑/○○는).

### 4.5 홈 화면 정책
- 게임 카드에 획득 별 뱃지, 상단에 총 별 카운터.
- **3별(마스터) 게임 카드는 자동으로 하단 정렬** (아직 깰 게임이 위로).
- 카드 맨 아래 **작은 초기화 버튼**(2단계 확인) — 모든 별·스티커 초기화(플레이어 선택은 유지).

## 6. 데이터 / 저장 정책

- 저장소: 브라우저 **localStorage** (서버 전송 없음)
- 키:
  | 키 | 내용 |
  |---|---|
  | `minigames:v1` | 게임별 `{ bestScore, stars(0~3), level }` |
  | `minigames:stickers` | 플레이어별 스티커 보유 `{ player: { stickerId: count } }` |
  | `minigames:player` | 선택된 플레이어(`가온이`/`시온이`) |
- 저장 실패(사생활 모드 등) 시 **인메모리 폴백**으로 동작(앱이 깨지지 않음).
- 초기화: 홈 하단 버튼으로 별·스티커 삭제 가능(기기 한정, 플레이어 선택은 유지).

## 7. 콘텐츠 / 라이선스 정책

- **자체 제작**: 효과음(합성), 숨은그림 SVG 장면, 산수/한글 데이터, 스티커(이모지) — 라이선스 이슈 없음.
- **외부 의존**:
  - 국기 이미지: `flagcdn.com` (외부 CDN, **온라인 필요**). 로드 실패 시 `ImageWithFallback`로 대체.
  - 일부 홈 카드 썸네일: Unsplash(기존), 산수/한글은 인라인 SVG, 국기 카드는 flagcdn.
- **아이 사진**: 선택 화면 썸네일용. 원본(고해상도)은 저장소에 커밋하지 않음(`.gitignore`), **256px·~9KB 썸네일만** 배포에 포함.
- 폰트: Google Fonts(Jua, Gaegu, Gamja Flower) — 손글씨풍.

## 8. 접근성 / 디바이스

- 큰 터치 타깃, 큰 글씨, 명확한 색.
- 마우스·터치·키보드(미로 화살표, 공룡 스페이스) 모두 지원.
- 드래그류는 텍스트 선택 방지(`user-select/touch-action: none`) 적용.
- 권장 환경: 태블릿(가로). 국기 게임은 인터넷 연결 시에만 국기 표시.

## 9. 배포 / 운영

- **저장소**: github.com/psynetJWlee/MiniGamesService
- **배포**: `main` 브랜치 푸시 → GitHub Actions(`deploy.yml`)가 `npm run build` 후 `dist/`를 GitHub Pages에 업로드.
- **SPA 라우팅**: `vite base = /MiniGamesService/`(빌드 시), 라우터 `basename` 일치, `404.html`(= index 복사)로 딥링크/새로고침 대응.
- **로컬 실행**: `start-dev.bat` 더블클릭 또는 `npm run dev`.

---

## 10. 알려진 제약 / 향후 과제

- 국기 게임은 외부 CDN 의존(오프라인 미작동) → 필요 시 `flag-icons` 패키지로 교체 가능.
- 별·스티커 기록은 **기기 로컬**이라 기기 간 동기화 없음(계정/서버 없음).
- 공룡 점프 충돌 판정은 단순화(유아 난이도).
- 향후 후보: 게임/스티커/장면 추가, 부모용 설정(사운드 토글), 효과음 on/off, 단어·문제 풀 확장.
