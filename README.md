# 가온이랑 시온이랑 놀자 (Mini Games Service)

우리 아이 첫 미니게임 — 유아·아동을 위한 한국어 교육용 미니게임 모음 웹앱입니다.
숫자·한글·관찰·추리 등 **능력별 17가지 놀이**를 통해 즐겁게 배우고, 게임을 클리어하면
별과 스티커를 모을 수 있습니다.

## ✨ 주요 특징

- **능력별 카테고리** — 17개 게임을 아이가 기르고 싶은 능력에 따라 5개 묶음으로 정리
- **플레이어 사용자화** — 기본 친구(가온이·시온이) 외에 **사진 + 이름으로 우리 아이 추가** 가능
- **별 & 스티커 보상** — 게임을 깰 때마다 스티커 1장, 별 3개(완벽)로 깨면 반짝이 스티커까지. **총 120종** 스티커 도감
- **화면에 꼭 맞는 UI** — 게임 화면은 가로·세로 어디서도 스크롤 없이 화면에 딱 맞게 (모바일·태블릿·데스크톱 대응)
- **즐거운 피드백** — 사운드 효과와 색종이(confetti) 축하 연출
- **설치 불필요·오프라인 진행 저장** — 모든 진행 상황(별·스티커·플레이어)은 브라우저 localStorage에 저장 (서버 없음)

## 🎮 게임 목록 (능력별)

| 카테고리 | 게임 |
|---|---|
| 🔢 **수리·계산력** | 산수 놀이 · 마트 장보기 · 시계 보기 · 누가 더 클까? · 풍선 터뜨리기 |
| 🧩 **사고·추리력** | 패턴 찾기 · 길 찾기 미로 · 직소 퍼즐 |
| 👀 **관찰·기억력** | 같은 그림 찾기 · 숨은 그림 찾기 · 색깔 몬스터 잡기 · 동물 소리 맞추기 |
| 🗣️ **언어·상식** | 한글 만들기 · 국기 맞추기 |
| 🤗 **교감·생활 놀이** | 동물 먹이 주기 · 동물 병원 놀이 · 공룡 점프 게임 |

## 👧 플레이어 & 진행 저장

- 기본 친구(가온이·시온이)는 항상 제공되며, 첫 화면의 **＋ 추가** 버튼으로 우리 아이를 직접 등록할 수 있습니다.
- 추가 시 디바이스에서 사진을 고르면 자동으로 작은 정사각형 이미지로 변환되어 저장됩니다.
- 별·스티커·게임 기록은 **플레이어 이름별로 따로** 관리됩니다.

진행 데이터는 다음 localStorage 키에 저장됩니다.

| 키 | 내용 |
|---|---|
| `minigames:player` | 현재 선택된 플레이어 |
| `minigames:customPlayers` | 추가한 커스텀 플레이어(이름·사진) |
| `minigames:v1` | 게임별 최고 별·점수·도달 단계 |
| `minigames:stickers` | 플레이어별 스티커 보유 현황 |

## 🧩 퍼즐 이미지 교체

직소 퍼즐의 그림은 `src/assets/puzzle/` 폴더의 이미지를 **자동 탐색**합니다.

- 지원 형식: `.jpg` `.jpeg` `.png` `.webp` (폴더 직속 파일만, 하위 폴더 제외)
- **정사각형에 가까운 이미지 권장** (퍼즐 칸이 사각형)
- 폴더가 비어 있으면 임시로 기본 사진이 사용됩니다.
- 이미지를 추가/삭제한 뒤에는 빌드(또는 개발 서버 재시작)하면 반영됩니다.

## 🛠️ 기술 스택

- **React 18** + **TypeScript** + **Vite 6**
- **Tailwind CSS 4** (스타일)
- **React Router 7** (라우팅)
- **Motion** (애니메이션) · **react-dnd** (드래그앤드롭) · **canvas-confetti** (축하 효과) · **lucide-react** (아이콘)

## 🚀 실행 방법

```bash
npm i        # 의존성 설치
npm run dev  # 개발 서버 실행
```

기타 스크립트

```bash
npm run build      # 타입체크 + 프로덕션 빌드
npm run typecheck  # 타입 검사만
npm run preview    # 빌드 결과 미리보기
```
