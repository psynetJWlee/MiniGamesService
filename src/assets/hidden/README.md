# 숨은 그림 찾기 이미지 폴더

**진짜 숨은그림찾기** — 물건들이 그림 속에 숨겨진 큰 일러스트 한 장 + 아래에 찾을
물건 목록. 지금은 저작권 없는 **샘플 일러스트(`sample-park.svg`)**로 동작합니다.

## 퍼즐 1개에 필요한 것

1. **일러스트 그림 1장** — 물건들이 그림 속에 자연스럽게 숨겨져(위장돼) 그려진 그림.
   - 형식: `.png` / `.jpg` / `.webp` / `.svg`
   - 가로:세로 비율은 자유(데이터의 `aspect`에 그 비율을 적어주면 됨).
   - ⚠️ **저작권 주의:** 블로그/시중 워크시트 이미지는 대부분 저작권이 있어요.
     본인이 만든/사용권이 있는(또는 CC0·퍼블릭도메인) 그림만 넣어주세요.

2. **각 물건의 숨은 위치(좌표)** + 목록에 쓸 **아이콘**(이모지 또는 윤곽 그림).

## 좌표 찍기 (핵심)

물건이 그림에 그려져 있으므로, "그 물건이 그림 어디에 있는지"를 좌표로 알려줘야 합니다.
개발 서버에서 **`/game/hidden?edit`** 로 열면, 그림을 클릭할 때마다 그 지점의 정규화 좌표
(`x`, `y` — 0~1)가 **화면 좌하단과 콘솔에 표시**됩니다. 숨은 물건을 클릭해 좌표를 읽어
`hiddenScenes.tsx`에 넣으세요.

## 데이터 연결 — `src/app/lib/hiddenScenes.tsx`

```ts
import myScene from '../../assets/hidden/farm.png';

export const PUZZLES = [
  {
    id: 'farm',
    title: '농장',
    image: myScene,
    aspect: 1000 / 700,           // 그림의 가로/세로 비율
    targets: [
      { id: 'spoon', name: '숟가락', icon: '🥄', x: 0.41, y: 0.62, r: 0.06 },
      // icon 대신 윤곽 이미지: { ..., iconSrc: spoonOutlineUrl }
      ...
    ],
  },
];
```

- `x`, `y`: 그림 속 물건 중심(0~1). `r`: 탭 허용 반경(그림 너비의 비율, 보통 0.05~0.08).
- `icon`: 목록 썸네일(이모지). 진짜 윤곽 그림을 쓰려면 `iconSrc`에 이미지 URL.

## 목록 아이콘(윤곽 그림)을 따로 쓰려면
찾을 물건을 **선(윤곽) 그림**으로 작게 만들어(투명 PNG/SVG) `iconSrc`로 지정하면
참고 이미지처럼 하단이 윤곽 리스트가 됩니다. 없으면 이모지로 표시됩니다.

> 이미지를 추가/교체한 뒤에는 빌드(또는 개발 서버 재시작)하면 반영됩니다.
