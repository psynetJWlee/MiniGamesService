# 몬스터 그림 폴더

"색깔 몬스터 잡기" 게임의 몬스터를 직접 만든 그림으로 바꿀 수 있어요.
여기에 이미지를 넣으면 **자동으로** 게임에 나타납니다. (폴더가 비어 있으면 기본
동그란 몬스터로 표시돼요.)

## 파일 이름 규칙 — 색깔로 짓기

게임은 "○○색 몬스터를 잡아요!"로 진행되므로, **파일 이름의 맨 앞에 색 이름**을
넣으면 그 색의 몬스터로 연결됩니다. (영문/한글 모두 가능)

| 색 | 파일 이름 예시 |
|---|---|
| 빨강 | `red.png` · `빨강.png` |
| 파랑 | `blue.png` · `파랑.png` |
| 노랑 | `yellow.png` · `노랑.png` |
| 초록 | `green.png` · `초록.png` |
| 보라 | `purple.png` · `보라.png` |
| 주황 | `orange.png` · `주황.png` |

### 한 색에 여러 마리(랜덤 등장)도 가능
이름 뒤에 `-`로 구분해서 번호/이름을 붙이면 같은 색의 여러 몬스터가 랜덤으로 나와요.

```
red-1.png   red-2.png   red-dragon.png   → 모두 "빨강" 몬스터
blue-slime.png                            → "파랑" 몬스터
```

## 권장 사양
- 형식: `.png`(투명 배경 권장) · `.webp` · `.svg` · `.jpg` · `.gif`
- 크기: **정사각형, 한 변 256~512px** 권장 (게임에서 약 96~112px로 표시)
- **배경이 투명**하면 어두운 게임 화면에 자연스럽게 떠 보여요
- 그림이 무슨 색인지 한눈에 보이게 (몬스터 뒤에 같은 색 후광이 깔립니다)

> 그림을 추가/교체한 뒤에는 빌드(또는 개발 서버 재시작)하면 반영됩니다.
> 이미지를 넣지 않은 색은 기본 동그란 몬스터로 표시됩니다.
