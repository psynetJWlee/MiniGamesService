# 퍼즐 그림 폴더

여기에 직소 퍼즐로 쓸 이미지를 넣으면 **자동으로** 게임에 나타납니다.

- 지원 형식: `.jpg` `.jpeg` `.png` `.webp`
- **정사각형에 가까운 이미지 권장** (퍼즐 칸이 사각형이라 비율이 많이 다르면 늘어나 보일 수 있음)
- 권장 크기: 한 변 600~1000px 정도 (너무 크면 용량↑ — 필요하면 리사이즈해서 넣기)
- 파일명은 자유 (한글/영문 모두 가능하지만 영문 권장)

이 폴더가 비어 있으면, 임시로 아이 사진(가온이/시온이)이 퍼즐 그림으로 사용됩니다.

> 이미지를 추가/삭제한 뒤에는 빌드(또는 개발 서버 재시작)하면 반영됩니다.
